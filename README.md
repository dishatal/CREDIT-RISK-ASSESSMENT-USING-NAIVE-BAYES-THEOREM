# Credit Risk Analysis Using Naïve Bayes

## Project Overview
This project applies the Gaussian Naïve Bayes algorithm to analyze a credit dataset and predict loan-related outcomes. The goal is to explore how machine learning models can assist in understanding financial data patterns.

## Dataset
The dataset contains information about loan applicants including:
- Age
- Job
- Housing
- Credit amount
- Loan duration
- Account information

## Tools Used
- Python
- Pandas
- Scikit-learn
- Seaborn
- Matplotlib
- Google Colab

## Model
The project uses the Gaussian Naïve Bayes classifier to train and predict outcomes based on applicant features.

## Evaluation Metrics
- Accuracy
- Confusion Matrix
- Precision
- Recall
- F1 Score

## Results
The model achieved approximately **33% accuracy** on the test dataset. The results show moderate performance due to dataset limitations and class imbalance.

## Project Pipeline
Data Cleaning → Feature Encoding → Train-Test Split → Model Training → Prediction → Evaluation
